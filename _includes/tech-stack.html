Tech Stack

Unity & C# / AI & ML / Python / Front-end / Flutter



저는 새로운 걸 배우고 도전하는 걸 무척 좋아해서 U-Day Seoul을 비롯한 유니티 게임 행사도 꾸준히 참여하고 있고, 메타버스 관련 기획으로 중소벤처기업부에서 진흥원장상을 받기도 했습니다.

최근에는 개발과 외국어에 대한 깊은 관심과 노력을 토대로 인천테크노파크에서 주관하는 ICT(인공지능/데이터/블록체인)분야 스타트업 네트워킹 프로그램에서 최종 4인에 선정되어 DUBAI GITEX 전시회에 통역/홍보/사전조사 등의 역할을 다양하게 수행하는 서포터즈 자격으로 참석할 예정입니다.

또한 졸업 이후 Unity를 이용해 각종 게임을 구현해보았고, IT 분야로 전환한 이래 꾸준히 다양한 프로그래밍 언어와 도구를 학습해왔습니다.

구체적으로는 C# 이외에도 Python, R, SQL, Excel, C++, java, Flutter, React Native, FE(HTML, CSS, javascript)등을 두루 다뤄보았고 문서 작성 및 디자인, 영상 편집, 3D 모델링(SketchUp, blender, Fusion360), 발표 등에 사용되는 툴도 시의적절하게 사용할 수 있습니다.

Python의 경우 Machine Learning Library를 활용하여 감성분석, 자연어처리, 금융 데이터 분석, 머신러닝, 딥러닝, 시각화 등을 구현할 수 있습니다. 특히 이전 회사에서 차량 흐름 데이터를 Edge PC에 접목시켜 스마트 교통 흐름 분석 및 자율 주행 관련 프로젝트를 진행해보았습니다.
